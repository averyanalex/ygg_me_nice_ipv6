# ygg_me_nice_ipv6
This go project generates nice ipv6 addresses such as ...aaaa:bbbb...., abab:cdcd , etc

example:
5 is a number of matching bytes and 1000000000 attempts

>ygg_nice_ipv6.exe 5 1000000000
found CPU cores: 12
